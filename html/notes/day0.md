## HTML Structure

Basic structure of every HTML page:

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

</body>
</html>

## Understanding

- <!DOCTYPE html> → tells browser this is HTML5
- <html> → root of the page
- <head> → contains meta info (not visible)
- <body> → contains all visible content

## Running HTML
