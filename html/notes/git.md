- Cloned my GitHub repository to local machine

## Basic Workflow

git add .

## Clone Repository

git clone <repo-link>
cd repo-name
code .

## Understanding

- Git = version control system
- GitHub = online repository
- VS Code = coding environment

## Common Commands

git status
git add .
git commit -m "message"
git push

## Mistakes I made

- Tried to run HTML file as command
- Was confused about folder structure

## Fix

-### Fix

- Use `start index.html` to open HTML
- Always go inside repo folder before pushing

### Key Learning

- One project = one repo
- Always organize folders properly
- Commit after every progress

## Important Workflow

Always before pushing:

git pull origin main --rebase
git push
